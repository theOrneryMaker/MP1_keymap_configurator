MP1 USER GUIDE

=== GESTURES ===
• Short tap encoder button (< 0.4s) → Open Menu 1 (Layer selection)
• Long press encoder button (≥ 1.5s) → Open Menu 2 (Utilities)
• Hold encoder button + turn clockwise → Toggle Jiggle (mouse wiggle)
• Hold encoder button + turn counter-clockwise → Enter Secret Layer

=== SECRET LAYER ===
• Accessed by hold + counter-clockwise turn
• Automatically exits after 15 seconds of inactivity
• All activity (keys, encoder, menus) resets the timer

=== USING THE CONFIGURATOR ===
1. Load your current keymap.json (optional)
2. Edit layers, keys, icons, encoder actions
3. Click "Generate keymap.json"
4. Click "Save keymap.json"
5. Copy the file to your MP1 macropad
6. Eject and reset – changes are live!

=== KEY TYPES ===
Website
• Full URL starting with http:// or https://
• Example: https://github.com

App (Windows only)
• Just type or paste the full path – win: prefix is added automatically
• Example: C:\Program Files\Google\Chrome\Application\chrome.exe

String
• Plain text that will be typed
• Perfect for emails, passwords, snippets

Key Combo
• Up to 3 keys from dropdowns
• Use LEFT_/RIGHT_ versions for side-specific modifiers
• Example: CONTROL + SHIFT + T

Consumer Control
• Media keys: PREV, PLAYPAUSE, NEXT, MUTE, VOL_DN, VOL_UP

User Defined
• Raw Python / CircuitPython code
• Examples:
  - Keycode.F13
  - (Keycode.CONTROL, Keycode.SHIFT, Keycode.ESCAPE)

=== ICON FIELD ===
• Max 4 characters (shown on OLED next to key)
• Uppercase recommended for best readability
• "---" = no icon

=== ENCODER ACTIONS ===
None – disabled
Volume – turn for volume
Zoom – turn for zoom in/out
Scroll – turn for page up/down
Custom – write your own dictionary
  • Also supports custom 4-character icon (e.g. MOUSE, VOL+, ZOOM)


=== DEFAULT & SECRET LAYERS ===
• Only one Default layer (boot layer)
• Only one Secret layer
• A layer cannot be both

=== QUICK START EXAMPLE ===
Want a media + app launcher layer?
1. Set Layer 0 as Default
2. Key 0 → App → C:\Program Files\Spotify\Spotify.exe (win: added automatically)
3. Key 1 → Consumer Control → PLAYPAUSE
4. Key 2 → Consumer Control → NEXT
5. Encoder → Volume
6. Generate → Save → Flash → Done!

Enjoy your fully customized MP1 macropad!
